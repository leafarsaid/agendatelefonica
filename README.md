# Agenda telefônica

Uma agenda simples de contatos telefônicos

## Instalação

Servidor MAMP, LAMP, WAMP com php 5.6 ou superior, com suporte a Sqlite 3.
Copiar os arquivos para seu DocumentRoot e executar no browser.
